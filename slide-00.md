# **คำอธิบายรายวิชา**

ศึกษาหลักการพัฒนาแอพพลิเคชันบนอุปกรณ์เคลื่อนที่  
โดยใช้ **React Native และ Expo** ครอบคลุม:

- Component-based Development  
- State & Props  
- Hooks  
- Layout & UX  
- การจัดการข้อมูล  
- การออกแบบ Navigation  

ผู้เรียนฝึกสร้างแอพต้นแบบใช้งานได้จริง  
รองรับงานด้านสุขภาพและสารสนเทศทางการแพทย์

---

# **วัตถุประสงค์รายวิชา (Course Objectives)**

1. อธิบายหลักการและสถาปัตยกรรมของ React Native  
2. ใช้องค์ประกอบ UI และ Hooks ได้อย่างเหมาะสม  
3. ออกแบบ UI/UX สำหรับ Mobile Application  
4. จัดการข้อมูลและสถานะของแอพ  
5. ออกแบบ Navigation และโครงสร้างหน้าจอ  
6. พัฒนาแอพต้นแบบที่ทำงานได้จริงผ่าน Expo
